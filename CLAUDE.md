# CorporateCareer richtlijnen

## Schrijfstijl (geldt voor alle teksten, site en communicatie)
- Gebruik nooit emoji's of smileys. Ook niet als icoontjes in HTML: gebruik inline SVG's.
- Gebruik nooit em-dashes of gedachtestreepjes. Herformuleer met een komma, dubbele punt of aparte zin.
- Geen verzonnen cijfers of claims (bezoekersaantallen, slagingspercentages, testimonials). Alleen verdedigbare feiten.

## Deployment
- corporatecareer.nl: ontwikkelen op branch, PR naar main, daarna gh-pages syncen via git archive vanaf main (gh-pages is de live branch).
- corporatecareer.be (repo corporatecareer-be): push naar main, GitHub Pages serveert main direct.
